## 从排序数组中删除重复项

给定一个排序数组，你需要在原地删除重复出现的元素，使得每个元素只出现一次，返回移除后数组的新长度。

不要使用额外的数组空间，你必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。

Go
```go
func removeDuplicates(nums []int) int {
    var i, j, length, length1 int
    length = len(nums)
    for i = 0; i < length; i ++ {
        for j = i + 1; j < length; j ++ {
            if (nums[i] != nums[j]) {
                length1++
                nums[length1] = nums[j]
                i = j - 1
                break
            }
        }
    }
    length1++
    nums = nums[:length1]
    return length1
}
```
